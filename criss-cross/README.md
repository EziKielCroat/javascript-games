## Steps to host

1. Make sure you have node and npm installed.
> node -v && npm -v 
2. Launch npm
> npm init --yes
3. (OPTIONAL) Installing eslint
> npm i eslint --save-dev --save-exact
4. Install required packages
> npm i socket.io express@4 path http
